Important Note to Designers and Developers

The CMS Kitchen Sink is a showroom for all CURRENTLY IMPLEMENTED features in the CMS. The Kitchen sink is NOT meant to be an accurate reflection of all component specs as they appear in Confluence, but rather an up-to-date representation of components which are already being used in production.

Developers should continuously add to the Kitchen Sink as they develop new components or make modifcations to existing ones. The appearance and behavior of components should reasonably reflect the specs defined in Confluence, with some notable differences: The UX/Design team uses pixel values in their designs to maintain consistency. Since Blazer uses rems as the base unit, it will not be possible to guarantee pixel-perfect accuracy, however the UX team specs are built on top of the core typography definitions which also form the basis for Blazer's layout engine. This will provide for a close match to the Confluence specs while maintaining consistent and automatic responsive behavior in all browsers.

Avenir

Designed by Adrian Frutiger in 1988, Avenir™ is a linear sans in the tradition of Erbar and Futura. Unlike Futura, Avenir is not purely geometric; It has vertical strokes that are thicker than the horizontals, an "o" that is not a perfect circle, and shortened ascenders. These nuances aid in legibility and give Avenir a harmonious and sensible appearance for both texts and headlines.

Avenir™ is available on most Macs and some Windows machines.

Helvetica

Helvetica is one of the most popular typefaces of the 20th Century. It is included in some form on almost all Macs and many Windows machines.

Arial

Arial is a sans-serif font included on all Windows machines. It is nearly ubiquitous, making it a good choice as a fallback font.

Sans-serif

In the unlikely event that none of the above fonts are available on a user's machine, Blazer will roll back to the default OS sans-serif typeface.
